hardshare
=========

**hardshare** is a system for sharing your hardware through the rerobots_
infrastructure.

Before you can share your robots, you must have a user account on rerobots.
The service is not yet generally available. If you are interested, `ask for
an account <https://rerobots.net/contact>`_.

The source code repository is at https://github.com/rerobots/hardshare
